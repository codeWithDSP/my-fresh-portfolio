# DSP Portfolio (static site)

This is a simple static portfolio built with **HTML + CSS + JavaScript**.

## How to run

- **Option 1 (quickest)**: open `index.html` directly in your browser.
- **Option 2 (recommended)**: run a local server (avoids some browser restrictions).

If you have Python:

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Suggested clean structure (next step)

Keep files organized like this:

```
/
  index.html
  styles.css
  script.js
  assets/
    img/
    icons/
```

Move images into `assets/img/` and update paths in `index.html`.
